## Quick start

```bash
degit radix-vue/shadcn-vue-nuxt-layer
```

## Usage

You can use these shortcut command to init, or add components

```bash
pnpm shadcn:init
# equivalent to npx shadcn-vue@latest init
```

 
```bash
npx shadcn:init add
# equivalent to npx shadcn-vue@latest add
```
